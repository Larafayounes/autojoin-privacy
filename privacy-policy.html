Privacy Policy

Effective Date: June 2026

Overview

AutoJoin Manager is a desktop add-in for Autodesk® Revit®. This policy explains what information is collected when you use the plugin and how it is handled. The short version: no data is ever sent to any server. Everything the plugin does stays on your machine.

Data We Do Not Collect

AutoJoin Manager does not collect, transmit, store, or share any personal data, usage data, model data, or analytics. There are no background processes, no internet connections, and no telemetry of any kind built into the plugin.

The plugin never communicates with any external server — not during installation, not during use, and not during license activation.

Local Storage

AutoJoin Manager stores two types of information locally on your machine only:

Trial period tracking — a timestamp recorded in your Windows AppData folder and registry, used solely to enforce the 7-day trial period. This information never leaves your machine.

Category settings — your last-used category priority list, saved in your Windows AppData folder so you don't need to reconfigure the plugin each time you open it. This information never leaves your machine.

License Activation

License activation is handled manually and entirely offline. When you purchase a license, you provide your Machine ID to the publisher by email. The publisher sends back an activation key. The key is validated locally on your machine with no server call required.

Your email address is used only to deliver your license key and is not shared with any third party.

Third-Party Services

AutoJoin Manager does not integrate with, or send data to, any third-party service, analytics platform, or cloud provider.

Children's Privacy

AutoJoin Manager is a professional software tool intended for use by adults in an architectural or engineering context. It is not directed at children under the age of 13 and does not knowingly collect any information from children.

Data Retention & Deletion

All data stored by AutoJoin Manager exists only on your local machine and is retained until you uninstall the plugin or delete the files manually.

All locally stored data — including trial period tracking, category settings, and license information — is automatically removed when you uninstall the plugin via Windows Settings → Apps → AutoJoin Manager → Uninstall.

Local data files are stored in your Windows AppData folder under a dedicated AutoJoin Manager subfolder. You may delete this folder manually at any time to remove all stored data without uninstalling the plugin.

Revoking Consent & Requesting Data Deletion

Since all data is stored locally on your machine and never transmitted to any server, you have full control over it at all times.

To delete all data stored by AutoJoin Manager, simply uninstall the plugin using Windows Settings → Apps → AutoJoin Manager → Uninstall. This removes all plugin files and locally stored data automatically.

To revoke consent or request confirmation that no personal data is held about you, contact the publisher at larafayounes@gmail.com. Since no personal data is stored on any server, there is nothing to delete remotely — but the publisher will confirm this in writing upon request.

Changes to This Policy

If this policy is updated, the revised version will be posted at this URL with an updated effective date. Continued use of the plugin after any change constitutes acceptance of the updated policy.
